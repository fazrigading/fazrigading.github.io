# Fazri Gading | Portfolio Source Code

This repository contains the source code for my personal portfolio website, hosted at `[https://fazrigading.github.io/portfolio](https://fazrigading.github.io/portfolio).

---

## Technical Architecture

* **Frontend:** Vite, React, TypeScript, and Tailwind
* **Hosting:** GitHub Pages
* **Deployment Path:** [https://fazrigading.github.io/portfolio](https://fazrigading.github.io/portfolio)

---

## Core Focus Areas

### Computer Vision & Deep Learning

* Implementation and evaluation of state-of-the-art object detection models ranging from YOLOv5 to YOLOv13, RT-DETR, and RF-DETR.
* Application of segmentation models like the Segment Anything Model (SAM) for complex agricultural and specialized datasets.

### Academic Research

* Developing robust benchmarks and writing technical manuscripts for precision agriculture and automated object detection systems.

### System & Tooling Development

* Maintaining upstream architectures for specialized game infrastructure and building native tools tailored for linux environments.

## Local Development

You can preview the portfolio locally by running a static file server.

```bash
# Clone the repository
git clone https://github.com/fazrigading/portfolio.git

# Enter the directory
cd portfolio

# Run a local Python server
python3 -m http.server 8000
```

Once running, navigate to `http://localhost:8000` in your browser.
